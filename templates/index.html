🗂 How to Create NocoDB Filtered Views

Cloud tier blocks view creation via API — here's how to do it manually in ~30 seconds:

  1. Open your base → app.nocodb.com → Employee Nirvana
  2. In the Views panel on the left, click + Add View → Grid
  3. Name it SDR Candidates → click + Add Filter → Top Role Fit is SDR
  4. Repeat for AE Candidates (filter: Top Role Fit is AE)
  5. Repeat for Researcher Candidates (filter: Top Role Fit is Researcher)
  6. Optional: Add a Kanban view grouped by Recommendation to visualise pipeline visually

✅ Once the Researcher JD is ready, share it here and every candidate will be scored against it immediately.

🤖 How to Score New Resumes with Alfred (Claude)
  1. Upload a PDF/DOCX above — it gets added to NocoDB with a preview
  2. Copy the extracted text from the preview box
  3. Open WhatsApp → message Alfred:
    Score this candidate for SDR/AE/Researcher: [paste extracted text]
  4. Alfred will return scores + analysis
  5. Update the NocoDB record manually, or tell Alfred: Update candidate Jane Smith: SDR Score 8.5, AE Score 6, Recommendation Strong Yes
  6. This page auto-refreshes scores from NocoDB ✅
